ngPopover

Introduction

An Angular 1.x directive for adding customisable popups to HTML elements.

Installation

npm install ng-popover

Usage

  1. Include angular-popover.js after loading AngularJS.
  2. Include angular-popover.css.
  3. Add the angular-popover module to your project:
    var app = angular.module('yourModuleName', ['angular-popover'])
  4. Add the angular-popover directive along with attributes - 'direction' and 'template':
    <div angular-popover direction="top" template="<your template path>" style="position: relative;">

Note

IMPORTANT: Keep in mind the element to which you are applying the directive has to be positioned - fixed, absolute or relative.

Demo

The popovers, by default, have a box shadow and a triangular tip. The style of the contents inside the popover is entirely customisable and upto you.
